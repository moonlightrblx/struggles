heres some basic tips for cyber security because people on the internet share too much data without realzing <3


# 1. do not not use google products if you're worried about security
google products are NOT built for security at all, they are meant to be easy to use for the average user

if you care about your data AT all then switch to proton for mailing + drive services + now they have a google sheets spreadsheet app and their own ai

for browser i would reccomend firefox or brave

# 2. use 2fa everywhere
people dont realize how easy it is to get into an acount without 2fa on it, its literally just a password login and sometimes an email verification code.

proton authenticator is my personal goto but there are better options.

# 3. use some sort of dns server that is not the default
i would recomend cloudflare dns servers 

# 4. vpn
a vpn isn't completely necessary but if you're doing anything on some sketchy websites or p2p sharing it should be.

if you don't care about being residential then use 1.1.1.1 (aka cloudflare warp) it barely reduces internet speed

# 5. social media 
social media is the number one way people are selling their data without realizing it. 

for social media obviously you cant avoid it, so disable anything in your settings about crossplatform sharing or advertising services. 

avoid using your main email (proton aliases are very good for this) so you can filter the spam from each company.

avoid using your real full name (for obvious reasons)

# 6. hardware
your hardware will never be secure because the NSA has a backdoor in every cpu made after 2008 (i may be wrong about the year but its been a while)

# 7. posting
basically if you post anything online there are MANY bots that will instantly scrape it and download it so nothing you post is private! (even if you post it as private)

strip metadata if you're really worried (https://www.metadata2go.com/delete-metadata)

heres common posts that people do and why you should avoid them

vacation posts (real-time or geotagged) -> burglary risk.

kids' names/schools/schedules -> predators.

home photos (street numbers, valuables in background) -> physical targeting.

birthdays, pet names, first car -> password reset/security question goldmine.

routines (gym times, dog walks) -> predictable patterns for tailing/scams
